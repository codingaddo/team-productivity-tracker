# Team Productivity Tracker Frontend

This is the Next.js (App Router) frontend for the Team Productivity Tracker monorepo.

## Tech stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Jest + React Testing Library

## Getting started

```bash
cd frontend
npm install
npm run dev
```

The app expects the following environment variable:

- `NEXT_PUBLIC_API_BASE_URL` – Base URL of the FastAPI backend (e.g. `http://localhost:8000`).

Run tests with:

```bash
npm test
```
